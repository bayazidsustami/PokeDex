# PokeDex
## Prerequisite
- Java 11
- Gradle 7.4
- Android Studio Dolphin (2021.3.1 Patch 1)

## Tech Stack
- Kotlin
- MVVM pattern
- Room 
- Retrofit2
- OkhttpClient
- Kotlin Coroutine + Flow
- Glide
- Shimmer by Facebook
- Mockk
- Espresso
